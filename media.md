This page collects the visual exhibits, diagrams, and associated media assets for the *If You Can't Beat 'Em* project. Assets are produced alongside the written pieces and updated as the argument develops. Print-resolution and downloadable copies of each exhibit are available on the [Downloads page]({{ '/downloads/' | relative_url }}).

<div class="exhibit">
  <p class="block-label">Exhibit 1 — The Divestiture Arc: Three Case Studies</p>
  <p>A comparison diagram tracing the institutional migration path of three legacy broadcast figures — Brian Williams (NBC → Netflix podcast), Chris Wallace (Fox News → CNN+ → independent podcast), and a third case TBD — against a timeline of broadcast audience decline. The diagram distinguishes between the adaptation path (format added, institution sustained) and the divestiture path (format adopted as institution contracts). Key inflection points marked: Rogan–Spotify deal (2020), CNN+ launch and collapse (2022), White House press pool expansion (2025–2026), Williams–Netflix debut (2026). <em>Asset not yet produced. Target format: horizontal timeline, two-column contrast layout, SVG with web and print resolution.</em></p>
</div>

<div class="exhibit">
  <p class="block-label">Exhibit 2 — Function Map: Broadcast Journalism vs. Podsphere</p>
  <p>A comparison table mapping journalism's democratic functions — common information environment, agenda-setting, credentialing of sources and claims, accountability journalism, public sphere participation — against the structural capacity of broadcast journalism (peak era), legacy broadcast (2026), and the podsphere (2026). Each cell is rated: performs well / performs partially / does not perform structurally / performs differently. Intended to make the "which functions survive?" question concrete for practitioners and to anchor the theoretical argument about function loss versus format adaptation. <em>Asset not yet produced. Target format: 5×4 comparison table, color-coded cells, SVG with accessible alt text.</em></p>
</div>

<div class="exhibit">
  <p class="block-label">Exhibit 3 — The Hybrid Media System: Infrastructure Layer</p>
  <p>A diagram illustrating the Chadwick hybrid media system model extended with the infrastructure-collapse account developed in the journal article. The base Chadwick model shows older and newer media logics interacting and mutually remediating. The extension adds an infrastructure layer beneath each node, showing the material and institutional substrate (licensing regimes, production economies, professional training institutions, distribution networks) that sustains each logic. The collapse is visualized as the substrate beneath the broadcast node contracting, with the broadcast logic degrading as its material supports thin. The podsphere node's substrate is shown as emergent but already institutionally developed (Spotify economics, podcast professional norms, platform distribution infrastructure). <em>Asset not yet produced. Target format: two-panel diagram — base Chadwick model, then extended model — SVG.</em></p>
</div>
